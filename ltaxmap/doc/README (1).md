# LTAX WebGIS - คู่มือการใช้งาน

## ภาพรวมระบบ
ระบบ WebGIS สำหรับจัดการข้อมูลที่ดิน รองรับการวาด แก้ไข แบ่ง รวม และจัดการแปลงที่ดิน

---

## ฟีเจอร์หลัก

### 🎨 การวาดและแก้ไข
- **➕ วาด**: วาดแปลงใหม่ พร้อมกรอกข้อมูล 9 ฟิลด์
  - zone, block, lot, parcel_code
  - survey, land_no, Mapsheet, land_type, Scale
  - คำนวณพื้นที่อัตโนมัติ (ไร่ งาน วา)

- **✏️ แก้ไข**: แก้ไขข้อมูลในฟิลด์
  - คลิกแปลง → กรอกข้อมูลใหม่
  - ไม่แตะ area, created_at, updated_at

### ✂️ แบ่งและรวมแปลง
- **✂️ แบ่งแปลง**: 
  1. กดปุ่มแบ่งแปลง
  2. คลิกเลือกแปลง (ขอบแปลงเป็นสีส้ม)
  3. วาดเส้นตัดผ่านแปลง
  4. แปลงจะแบ่งเป็น 2 ส่วน

- **🔗 รวมแปลง**:
  1. กดปุ่มรวมแปลง
  2. คลิกเลือกแปลง 2+ แปลง (เป็นสีน้ำเงิน)
  3. ยืนยันการรวม
  4. ได้แปลงใหม่รวมกัน

### 💾 บันทึกและจัดการ
- **💾 บันทึก**: บันทึกข้อมูลไปยัง server (PHP)
  - เครื่องบ้าน: ใช้งานได้ ✅
  - เครื่องออฟฟิศ: มีปัญหา config (ใช้ Export แทน)

- **💾 Export**: ดาวน์โหลด GeoJSON
  - เลือก layer จาก dropdown
  - หรือ export ทั้งหมด

- **📂 Import**: โหลดไฟล์ GeoJSON เข้ามา
  - รองรับ Standard GeoJSON (WGS84)
  - รองรับ LTAX format (nested structure)
  - สร้าง layer แยก (สีส้ม) สำหรับเทียบข้อมูล

- **🗂️ จัดการ Layer**: จัดการ imported layers
  - `1-9` = เปิด/ปิด layer
  - `D1-D9` = ลบ layer
  - `A` = ลบทั้งหมด

### 🔍 ค้นหา
- ค้นหาจาก parcel_code, zone, block
- Highlight แปลงที่เจอ (สีแดง)
- Zoom ไปยังแปลงอัตโนมัติ

### 🔄 Reset
- ยกเลิกทุกโหมด
- ล้างการเลือก
- รีเซ็ตสีทั้งหมด

## การใช้งาน

### Workflow ทั่วไป
1. **Import** ข้อมูลเก่าเข้ามาเทียบ (layer สีส้ม)
2. **วาด** แปลงใหม่ใน layer หลัก (สีเขียว)
3. **แบ่ง/รวม** แปลงตามต้องการ
4. **บันทึก** หรือ **Export**
5. **ปิด/ลบ** imported layer

### Tips สำหรับแปลงชิดกัน
- **Zoom ใหญ่มากๆ** ก่อนคลิกเลือกแปลง
- คลิกที่ขอบแปลง (ไม่ใช่กลาง)
- ถ้ากดผิด → กด Reset แล้วเริ่มใหม่

---

## ข้อกำหนดไฟล์

### Import (รับได้)
- ✅ GeoJSON WGS84 (EPSG:4326)
- ✅ LTAX format (มี LocationGeospatial wrapper)
- ❌ UTM (EPSG:24047/24048) → ต้องแปลงใน QGIS ก่อน

### Export (ออกเป็น)
- GeoJSON WGS84
- ชื่อไฟล์: `layer_name_YYYY-MM-DD.geojson`

## ปัญหาที่พบบ่อย

### 1. Save ไม่ได้ที่ออฟฟิศ
**สาเหตุ**: PHP config/version/permission ต่างกัน  
**แก้**: ใช้ Export แทน → แล้วนำไฟล์ไป replace ใน `data/parcel.geojson`

### 2. Import แล้วหาย
**สาเหตุ**: พิกัด UTM แทน WGS84  
**แก้**: แปลงใน QGIS ก่อน
1. เปิดไฟล์ UTM
2. Right click → Export → Save Features As
3. CRS: EPSG:4326
4. Save

### 3. คลิกแปลงไม่ได้ (แปลงชิดกัน)
**แก้**: 
- Zoom ใหญ่มากๆ
- คลิกช้าๆ ที่ขอบแปลง
- ใช้ค่า fillOpacity: 0.01 (ใน config.js)

### 4. Split/Merge ไม่ทำงาน
**แก้**: 
1. กดปุ่ม Reset
2. Refresh (Ctrl+F5)
3. ลองใหม่

## เทคนิคการใช้งาน

### แปลงพิกัด UTM → WGS84
**ใน QGIS:**
1. เปิดไฟล์ GeoJSON (UTM)
2. Right click layer → Export → Save Features As
3. Format: GeoJSON
4. CRS: **EPSG:4326 - WGS 84**
5. Save

### เทียบข้อมูลเก่า-ใหม่
1. Import ข้อมูลเก่า → layer สีส้ม
2. วาดแปลงใหม่ → layer สีเขียว
3. เปรียบเทียบความแตกต่าง
4. Export ข้อมูลใหม่
5. จัดการ Layer → ลบ imported layer

### แบ่งแปลงที่ชิดกัน
1. **Zoom** ใหญ่มากๆ (จนเห็นแปลงชัดเจน)
2. กด **แบ่งแปลง**
3. **คลิกช้าๆ** ที่ขอบแปลง
4. รอจนเป็น**สีส้ม** (ยืนยันว่าเลือกถูก)
5. วาดเส้นตัด
